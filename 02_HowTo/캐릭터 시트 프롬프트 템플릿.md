# 캐릭터 시트 프롬프트 템플릿

> ChatGPT(GPT-4o) 이미지 생성으로 다중 뷰 캐릭터 시트를 만드는 복붙용 틀. `{ }` 슬롯을 채워 사용함. 원리는 [[캐릭터 시트]]·[[캐릭터 일관성]] 참고. **GPT 기반 생성이므로 작성 후 Codex 검수를 거침**(§사용법 4).

## 사용법 (프로세스)
1. 아래 템플릿의 `{ }`를 채움. 색은 가능하면 **헥스값**으로 고정(단, 정확한 헥스 재현은 보장 안 됨) (출처: src-openai-cookbook-imggen).
2. **크기 명시** — gpt-image-1 계열은 `1024x1024`·`1536x1024`(가로 3:2)·`1024x1536`(세로 2:3)만 지원함. **정확한 16:9는 gpt-image-2에서만 가능** (출처: src-openai-api-imggen, src-openai-cookbook-genimg).
3. 한 장에 뷰를 과하게 넣지 않음(객체 수 ~10~20 한계) → **전신 4뷰 + 헤드샷 분리 생성** (출처: src-openai-cookbook-imggen, 검수: Codex).
4. **Codex 검수** — 완성 프롬프트를 Codex(GPT 계열)로 검수해 GPT 이미지 생성 적합성을 확인함. 이미지 생성기가 GPT이므로 같은 계열 모델이 판정에 적합함 → [[프롬프트 검수 워크플로]].
5. 1차 생성 후 같은 대화에서 **소폭 보정**을 반복함 (출처: src-openai-cookbook-imggen).

## 템플릿 — 전신 4뷰 시트

```
Create a character reference sheet (model sheet) on a plain neutral gray background, even studio lighting. Generate at size 1536x1024 (landscape).

CHARACTER: {캐릭터명}, {나이/종족}, {성별}.
BUILD: {체형/키/비율}.
FACE: {얼굴 특징}, hair {헤어 스타일/색}.
OUTFIT: {의상 상세}. MATERIALS: {재질, 예: 천/가죽/방탄 소재}.
COLOR PALETTE: {주요 색 + 헥스값}.

LAYOUT — create exactly 4 full-body views only, evenly spaced in one horizontal row:
1) front view
2) back view
3) left side view
4) right side view

POSE — neutral A-pose for all views: arms slightly away from torso, legs straight, feet shoulder-width. Keep the pose identical across all four views.

Each view shows one complete full-body figure. Do not merge views, do not add extra characters, no expression variations, no props unless specified. Do not crop feet, hands, hair, or equipment.

This is a consistency reference sheet, not an action frame: neutral posture only, no motion blur, no dramatic angle, no action effects.
Keep identical facial features, proportions, outfit, and color palette across ALL views. Do NOT redesign the character. Line style: {스타일, 예: clean anime cel-shading}.
Do not add text labels inside the image; keep the background blank and clean.
```

## 헤드샷 (별도 생성)

```
Create a separate close-up head reference of the SAME character, size 1024x1024, neutral expression, same hair, face, collar, and color palette as the character sheet. No text labels.
```

## 변형 슬롯

| 목적 | 추가/교체 문구 |
|------|----------------|
| 기존 확정 캐릭터 고정 | "Reference — Image 1 is the approved design. Recreate the SAME character only; preserve face shape, hairstyle, outfit silhouette, colors, accessories. Do not redesign." (앵커 이미지, 출처: src-openai-cookbook-imggen) |
| 표정 시트 | "expression sheet: {기쁨/분노/놀람/...} headshots of the same character" (별도 이미지로) |
| 의상 변형 | "outfit variations: {일상복/전투복/...}, same face and body" (별도 이미지로) |
| 그리드 레이아웃 | "Generate at size 1024x1024. Arrange the 4 views in a clean 2x2 grid." |

> ⚠️ **액션 포즈(combat stance 등)는 기준 시트에 섞지 않음** — 기준이 흔들려 후속 샷 참조성이 떨어짐. 액션은 [[스토리보드 프롬프트 템플릿]]에서 다룸 (검수: Codex).
> ⚠️ 다중 뷰 레이아웃·포즈 표준은 **커뮤니티 관행**(공식 템플릿 아님). T-pose/back/side 등은 모델이 대체로 이해하나 CAD식 정렬은 보장 안 됨 (출처: src-selfielab-charsheet, src-bigprompthub-turnaround, 검수: Codex).
