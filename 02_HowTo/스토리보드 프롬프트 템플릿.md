# 스토리보드 프롬프트 템플릿

> 캐릭터 시트를 참조로 첨부해 각 샷 이미지를 만드는 복붙용 틀. 산출 이미지는 [[Seedance]] I2V의 **첫 프레임**이 됨. 원리는 [[스토리보드]]·[[이미지 프롬프트 원리]] 참고. **GPT 기반 생성이므로 작성 후 Codex 검수를 거침**(§사용법 5).

## 사용법 (프로세스)
1. 캐릭터 시트를 **참조 이미지로 첨부**하고 프롬프트에서 인덱스로 지칭("Image 1"). 시트가 다중 뷰면 **어느 뷰를 기준**으로 삼을지 명시 (출처: src-openai-cookbook-imggen, 검수: Codex).
2. **크기 명시** — gpt-image-1 계열은 `1536x1024`(가로)·`1024x1536`(세로)·`1024x1024`만 지원. 시네마스코프(2.39:1)는 직접 생성 불가 → 가로로 생성 후 **후처리 center-crop** (출처: src-openai-api-imggen, src-openai-cookbook-genimg, 검수: Codex).
3. 한 샷은 **주 캐릭터 1명 + 필수 소품 1~2개**로 단순화(디테일 흐트러짐 방지) (검수: Codex).
4. 정지 포즈는 **다음 움직임이 시작 가능한 안정 포즈**로 — 영상 첫 프레임 품질 직결 → [[영상 프롬프트 원리]].
5. **Codex 검수** — 완성 프롬프트를 Codex(GPT 계열)로 검수해 GPT 이미지 생성 적합성을 확인 → [[프롬프트 검수 워크플로]].

## 템플릿 — 단일 샷

```
Storyboard frame. Generate at size {1536x1024 (16:9 video) / 1024x1536 (9:16 video)}.

Reference — Image 1 is the approved character sheet of {캐릭터명}. Use the front/3-4 views as the FACE reference and the full-body views for OUTFIT details. Preserve the same face shape, hairstyle, outfit silhouette, colors, and key accessories. Do NOT redesign, age up/down, change hairstyle, or swap costume pieces.

SHOT TYPE: {wide / medium / close-up}.
CAMERA ANGLE: {eye-level / low-angle / high-angle}.
STYLE: {예: cinematic anime, clean cel shading}. Apply consistently, but PRIORITIZE character accuracy over stylization.
COMPOSITION: {화면 내 캐릭터 위치, 시선 방향, foreground/midground/background 배치}.
SETTING: {장소/배경 디테일}.
LIGHTING / MOOD: {조명, 예: soft diffuse / golden hour / high-contrast}, {분위기}.
ACTION (stable frozen pose): {정지 동작 + 소품 사용}. Physically balanced, readable as a single still: clear hands, feet, face direction, no duplicated/cropped limbs.

This image will be the FIRST FRAME for image-to-video. Make it a clean, stable starting pose before motion begins: no motion blur, no speed lines, no impact/smear frames.
Compose with safe margins for video crop.
```

## 샷 명세 빠른 표

| 요소 | 선택지(예) |
|------|-----------|
| 샷 타입 | wide / medium / close-up / extreme close-up |
| 앵글 | eye-level / low-angle(위압) / high-angle(취약) / top-down |
| 조명 | soft diffuse / golden hour / high-contrast / backlit |
| 크기→비율 | `1536x1024`(16:9 영상용) / `1024x1536`(9:16 영상용) / 시네마스코프는 가로 생성 후 crop |

> ⚠️ 샷 타입·앵글 의미는 일반 영화 지식임. 공식 가이드는 모델이 용어에 반응함만 확인 (출처: src-openai-cookbook-imggen).
> ⚠️ 클로즈업은 의상 일관성 확인이 어렵고 스타일이 얼굴 정체성을 덮기 쉬움 → 얼굴 근처에 의상/장비 단서 1개를 명시 (검수: Codex).

## 스타일 앵커 (선택)
- 액션·전투 샷은 작화 스타일을 `STYLE:` 슬롯에 명시: [[이타노 서커스]](고속 추격·미사일 군무) · [[카나다 스타일]](광각 임팩트·이펙트).
- 컷별 스타일 지정·복붙 키워드는 [[스타일 적용 규칙]] 참고.
- ⚠️ 강한 스타일 앵커는 캐릭터 정체성을 덮을 수 있음 → "Style must not override the reference character; keep facial identity, hair, outfit, palette from Image 1." 한 줄 추가 (검수: Codex).
