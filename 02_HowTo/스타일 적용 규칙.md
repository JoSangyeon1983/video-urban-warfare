# 스타일 적용 규칙 (이타노/카나다)

> 사용자가 컷/씬별로 작화 스타일을 지정("이 컷은 [[이타노 서커스]], 저 컷은 [[카나다 스타일]]")하면, **아래 고정 절차로 프롬프트에 적용함.** 매번 즉흥 작성이 아니라 동일 출처·동일 변환을 따름.

## 적용 절차 (지정 → 키워드 → 삽입 → 가드)
1. **스타일 식별** — 각 컷/씬에 어떤 스타일이 지정됐는지 확인. 지정 없는 컷은 스타일 앵커를 넣지 않음(기본 룩 유지).
2. **단계 확인** — 지금 만드는 게 **스토리보드 이미지(ChatGPT)**인지 **영상(Seedance)**인지 구분. 단계별로 키워드가 다름.
3. **키워드 인출** — 아래 §치트시트(또는 개념 페이지의 "프롬프트 적용" 섹션)에서 해당 스타일·단계의 키워드를 꺼냄.
4. **템플릿 삽입** — [[스토리보드 프롬프트 템플릿]] 또는 [[Seedance 모션 프롬프트 템플릿]]의 `STYLE:` 슬롯에 삽입.
5. **가드 적용** — §주의 규칙(캐릭터 정체성 우선, 카메라/주체 무빙 분리 등)을 반드시 붙임.
6. **(선택) 근거 확인** — 새 키워드 도출·검증이 필요하면 [[이타노 서커스 연구 정리]] / [[카나다 스타일 연구 정리]]로 드릴다운.

## 치트시트 (복붙용)

### 이타노 서커스 — 고속 추격·미사일 군무
- **스토리보드(ChatGPT) STYLE 슬롯:**
  ```
  extreme foreground missile with wide-angle distortion, receding missile trail lines for depth, dynamic three-point perspective, high-speed dogfight
  ```
- **Seedance Style/모션:**
  ```
  tracking shot locked to the subject, missiles swarming with curving smoke trails, motion blur, strong sense of speed
  ```

### 카나다 스타일 — 광각 임팩트·이펙트
- **스토리보드(ChatGPT) STYLE 슬롯:**
  ```
  extreme foreground limb or weapon, wide-angle lens distortion, low-angle charge, frozen impact pose, petal-shaped explosion, curved energy beam
  ```
- **Seedance Style/모션:**
  ```
  hold on a strong key pose, brief explosive move, petal-shaped burst as a separate effect beat; preserve composition and colors
  ```

## 주의 규칙 (가드 — 항상 적용)
| 상황 | 규칙 |
|------|------|
| 스타일 vs 캐릭터 | 강한 스타일 앵커가 얼굴·의상 정체성을 덮을 수 있음 → "Style must not override the reference character; keep facial identity, hair, outfit, palette from Image 1." 추가 (출처: src-openai-cookbook-imggen) |
| Seedance 공통 | 카메라 무빙과 주체 무빙을 **분리**해 기술. 한 컷에 주 동작 1개+카메라 무빙 1개 (출처: src-fal-learn-2-0) |
| 이타노 + Seedance | "fast" 남용 금지 — 빠른 카메라+빠른 컷+복잡 장면은 아티팩트 유발. 핵심 컷에만 강하게 (출처: src-fal-learn-2-0) |
| 카나다 + Seedance | 포즈-투-포즈는 영상모델 보간과 상충 → **키 포즈 컷을 여러 장 생성해 짧은 클립으로 잇고**, 이펙트는 별도 컷으로 (출처: src-kanada-style-report) |

## 동작 예시
입력: "1번 컷은 이타노 서커스, 2번 컷은 카나다 스타일, 둘 다 Seedance용."
→ 1번: Seedance 템플릿 + 이타노 Seedance 키워드 + 카메라/주체 무빙 분리 + "fast" 절제.
→ 2번: Seedance 템플릿 + 카나다 키워드 + 키 포즈 컷 분할 안내 + 이펙트 별도 컷.
