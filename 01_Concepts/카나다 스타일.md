# 카나다 스타일 (Kanada Style / 金田系)

> 카나다 요시노리가 정립한 작화 문법. **광각 왜곡 공간 · 포즈-투-포즈 비연속 타이밍 · 물리보다 화면 충격을 우선하는 이펙트 · 컷 내부 카메라까지 연기시키는 액션 연출**의 총체임 (출처: src-kanada-style-report). 본 프로젝트의 **임팩트·이펙트 샷 스타일 앵커**로 씀.

## 핵심 특성

| 층위 | 특징 |
|------|------|
| 작화 | 전경 파츠(주먹·총구·메카) 과감 확대, 후경 압축, 실루엣 우선("멀티 퍼스펙티브") (출처: src-kanada-style-report) |
| 타이밍 | 1·2·3·4·6코마를 섞는 "乱れ打ち", **정지-폭발-정지-잔향** 파형, 예비동작 길게·본동작 생략·임팩트 정지 (출처: src-kanada-style-report) |
| 이펙트 | 곡선형 빔("金田ビーム"), 꽃잎/파열형 폭발("金田爆発"), 과장된 반사광("金田光り") — 이펙트가 배우처럼 연기 (출처: src-kanada-style-report) |
| 액션 연출 | 저각도 돌진, 전경을 가르는 무기, 충돌 직전 급클로즈업, 포즈 완결성 기준 컷 분할 (출처: src-kanada-style-report) |

## 전형적 타이밍 구조 (보고서 재구성 모델)
1. 예비동작 4~8f (실루엣 고정)
2. 본동작 2~4f (중간 생략·점프)
3. 충돌 정지 2~3f (포즈 강조)
4. 후행 이펙트 3~6f (파편·속도선 분리)
5. 잔향 4~8f (흔들림·페이드)

(출처: src-kanada-style-report)

> ⚠️ 위 노출 프레임 수와 사례별 타임스탬프는 보고서가 **추정치/분석 모델**로 명시한 값임(블루레이 실측 전사 아님) (출처: src-kanada-style-report).

## 프롬프트 적용 (본 파이프라인 연결)
- **스토리보드 이미지**: "extreme foreground limb/weapon, wide-angle lens distortion, low-angle charge, frozen impact pose, petal-shaped explosion, curved energy beam" → [[스토리보드 프롬프트 템플릿]].
- **Seedance 모션**: 포즈-투-포즈는 영상모델의 보간과 상충하므로, **키 포즈 컷을 여러 장 생성→짧은 클립으로 잇는** 방식 권장. 이펙트는 별도 컷으로 → [[Seedance 모션 프롬프트 템플릿]].
- 정지화 한 장은 어색해도 연속에서 명쾌 — 멈춘 프레임이 아니라 **이동 중 실루엣 읽힘** 기준으로 설계 (출처: src-kanada-style-report).
